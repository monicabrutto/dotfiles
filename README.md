# dotfiles
user dotfiles. Created in CEG2410

### Installs 
*dnsutils (to get dig)
* nmap
* anaconda
* greed
* vundle

### Configures 
* gitconfig
    * name
    * email
    * vim
    * autocorrect
* bashrc
    * aliases
        * clear
        * exit
    * extract function to easliy extract files
* ssh keys
    * sets up ssh keys for fry.cs.wright.edu login
* vim
    * changes color scheme
    * adds fugitive plug in
